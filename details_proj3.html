Application de location de voiture

DESCRIPTION

Développement d’une application web de gestion de location de voitures permettant aux utilisateurs de consulter les véhicules disponibles, effectuer des réservations et gérer les locations. L’application permet également à l’administrateur de gérer les voitures, les clients et les réservations
